# Side Project I

Side Project Part I - 登入介面\
2024/8~仍在進行\
主要使用：ReactJS、Node.js、Express.js、passport-jwt\
資料庫：、PostgreSQL\
介面設計參考：AsmrProg\
原設計是使用html+CSS+JS撰寫，本計畫則以ReactJS重新編寫並配合Node.js、Express連結PostgreSQL資料庫來進行帳號密碼的驗證，而授權(authorization)程序則以JWT來進行。
